# Comparison of the Efficiency for Predicting House Price in Bangkok Metropolitan Region

Data Limitations

The dataset used in this study has several limitations that may affect the prediction performance.

First, the price variable has a very wide range, which may introduce variability and reduce the stability of the prediction models.

Second, the number of observations in the dataset is relatively small. A limited sample size may reduce the ability of the models to generalize and capture the true relationship between variables.

Additionally, the variable year built could not be included in the analysis due to a large number of missing values from the source website. As a result, this potentially important factor could not be used in the prediction models.

These limitations should be considered when interpreting the results of this study.
1.Project Overview

This project aims to compare the performance of different predictive models for estimating house prices in the Bangkok Metropolitan Region.
The study analyzes housing data collected from an online real estate platform and evaluates which model provides the most accurate prediction.

The results help identify important factors affecting house prices and determine the most efficient predicting technique.

2.Objective
To analyze factors affecting house prices in Bangkok Metropolitan Region
To build predictive models for house price estimation
To compare the efficiency of different prediction techniques
To identify the model with the highest prediction accuracy

3.The dataset was collected from an online real estate website (e.g., DDproperty).

Example variables include:

House price
Property type (Detached house / Townhouse)
Land area
Usable area
Number of bedrooms
Number of bathrooms
Location
Other housing characteristics

4.Methodology

The analysis consists of the following steps:
Data collection and cleaning
Exploratory data analysis (EDA)
Model development
Model evaluation and comparison

5. Prediction Models

The following models were used for predicting house prices:
Linear Regression
Artificial Neural Network: ANN
Random Forest
Extreme Gradient Boosting

6.Evaluation Metrics

The performance of each model was evaluated using:

MAE (Mean Absolute Error)
RMSE (Root Mean Squared Error)
R² (Coefficient of Determination)

The model with the lowest error and highest R² is considered the best-performing model.

7.Tools Used
Python / SPSS 
Excel
webscraper.io

8.Key Findings
Location and usable area are the most influential factors affecting house prices.
Some models perform better in predicting nonlinear relationships.
The comparison shows that randomforest regressor provides the highest prediction accuracy.
